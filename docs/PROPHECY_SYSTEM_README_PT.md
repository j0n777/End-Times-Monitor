# 📖 SISTEMA DE INTELIGÊNCIA PROFÉTICA

## Visão Geral

O **End Times Monitor** agora possui um **sistema abrangente de catalogação de profecias bíblicas** que monitora sinais dos tempos finais de forma sistemática e baseada em dados.

## 📚 Arquivos Principais

### 1. `PROPHECY_STUDY.md`
**Estudo Teológico Completo** em Português com análise detalhada de:
- Profecias de Daniel (4 Impérios, 70 Semanas, Tempo do Fim)
- Profecias de Isaías (Israel, Damasco, Egito)
- Profecias de Ezequiel (Gog e Magog, Ossos Secos, Terceiro Templo)
- Profecias de Zacarias (Jerusalém, Monte das Oliveiras)
- Profecias de Joel (Espírito Santo, Sinais Cósmicos)
- Discurso do Monte das Oliveiras - Jesus (Mateus 24, Marcos 13, Lucas 21)
- Apocalipse (7 Selos, Marca da Besta, Armagedom, Milênio)
- Profetas Menores (Sofonias, Amós, Miquéias, etc.)
- **⚠️ Profecias Islâmicas vs. Bíblicas** (O Grande Engano)

### 2. `prophecyData.ts`
**Banco de Dados TypeScript** com:
- **60+ profecias catalogadas** com status, categoria, evidências modernas
- Sistema de filtros por categoria e status
- Níveis de alerta (CRITICAL, HIGH, MEDIUM, LOW)
- Evidências modernas documentadas
- Dicas de monitoramento

### 3. `components/ProphecyIntel.tsx`
**Interface Visual Interativa** com:
- Timeline profético navegável
- Filtros por categoria e status
- Cards expansíveis com evidências e monitoramento
- Estatísticas em tempo real
- Design tático responsivo

## 🛠️ Como Usar o Sistema

### 1. Visualização Interativa
```
End Times Monitor → Menu "PROPHECY" → ProphecyIntel
```

### 2. Filtros Disponíveis
- **Por Categoria**: Daniel, Isaías, Ezequiel, Jesus, Apocalipse, etc.
- **Por Status**: Cumpridas, Em Progresso, Pendentes
- **Por Alerta**: Crítico, Alto, Médio, Baixo

### 3. Detalhes de Cada Profecia
Clique em "Ver Evidências e Monitoramento" para:
- **Evidências Modernas**: Eventos atuais cumprindo profecia
- **Dicas de Monitoramento**: O que observar no noticiário

### 4. Integração com Mapa
Eventos do mapa (conflitos, desastres) possuem campo `propheticReference`:
```typescript
{
  id: 'e1',
  title: 'Golan Heights Troop Movement',
  propheticReference: 'Jeremiah 49:23'
}
```

## ⚠️ Avisos Importantes

### 1. **Não Marcar Datas**
> "Mas daquele dia e hora ninguém sabe" (Mateus 24:36)

O sistema NÃO prevê datas específicas, apenas monitora cumprimento de sinais.

### 2. **Interpretação Bíblica**
As interpretações são baseadas em **escatologia cristã evangélica**. Outras tradições podem ter visões diferentes.

### 3. **Profecias Islâmicas**
A seção sobre Islã representa a **perspectiva cristã** de que Mahdi/Isa podem ser engano satânico (Anticristo/Falso Profeta). Não é ataque ao Islã, mas análise escatológica comparada.

### 4. **Verificação de Fontes**
Todas evidências modernas devem ser verificadas. O sistema fornece pistas, não verdades absolutas.
